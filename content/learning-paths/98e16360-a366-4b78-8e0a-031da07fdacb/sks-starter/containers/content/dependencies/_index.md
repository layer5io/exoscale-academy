### Installing Dependencies ...
... on Ubuntu

```bash
# Install repository
$ curl -fsSL https://deb.nodesource.com/setup_current.x | sudo -E bash –sudo apt-get update
# Install NodeJS
$ sudo apt-get install -y nodejs
# Install Dependencies of the app
$ npm install
```
